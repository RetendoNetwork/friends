# friends
- Friends Server Replacement for Retendo Network.

## Usage
- Install Python, Git and Docker.
- Clone the repository with `git clone https://github.com/RetendoNetwork/friends.git`.
- Rename the file `example.env` to `.env`.
- Build the friends server image with `docker build -t friends .`.
- And run the server without the bug error `docker run -t friends`.
